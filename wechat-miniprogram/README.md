# X健身计数

面向力量训练的微信小程序，用于控制组间休息并记录每组训练数据。

## 当前功能

- 记录动作、重量、次数或计时时长，并按天保存在本机。
- 完成本组后自动进入独立休息界面，倒计时切到后台后仍按真实时间恢复。
- 农夫行走支持左右手分别开始、结束和重测，并保存两侧时间戳。
- 仅扫描标准 BLE 心率服务并订阅 `0x2A37`，断连后立即清除过期实时值。
- 分块保存设备发送的完整原始心率样本；存储接近上限时，由用户确认是否只清理 30 天前的原始心率。
- 动作库、常用模板、今日统计和历史记录。

## 运行

1. 打开微信开发者工具。
2. 导入当前目录。
3. 点击“编译”查看模拟器；蓝牙功能需要在 iPhone 上使用“预览”或“真机调试”。
